# Azure Landing Zone using Terraform

## Project Overview

This project demonstrates how to deploy an Azure Landing Zone using Terraform following Infrastructure as Code (IaC) best practices.

## Technologies Used

- Microsoft Azure
- Terraform
- Azure Virtual Network
- Network Security Groups
- Azure Key Vault
- Azure Storage Account
- Azure App Service
- Azure Monitor
- Azure DevOps
